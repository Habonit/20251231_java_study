# 연습 문제 1: 기본형과 참조형의 차이

## 난이도: 기초

## 학습 목표

- 기본형(Primitive Type)과 참조형(Reference Type)의 차이를 이해한다
- int, long, double 등 기본형의 특징을 익힌다
- 기본형에는 실제 값이, 참조형에는 참조값이 저장됨을 이해한다

---

## 문제 설명

기본형 변수와 참조형 변수의 값 복사 방식이 어떻게 다른지 확인하는 프로그램을 작성하라.

**핵심 개념**:
- 기본형: 변수에 **실제 값**이 저장된다. 값을 복사하면 독립적인 복사본이 생긴다.
- 참조형: 변수에 **참조값(메모리 주소)**이 저장된다. 복사하면 같은 객체를 가리킨다.

---

## 요구 사항

1. `Data.java`에 `Data` 클래스를 정의하라
   - value (int): 정수 값

2. `Main.java`에서:
   - 기본형 변수 복사 실험: int 변수를 복사 후 원본 변경
   - 참조형 변수 복사 실험: Data 객체를 복사 후 원본 변경
   - 각각의 결과를 출력하여 차이점 확인

---

## 예상 출력

```
=== 기본형 복사 실험 ===
a = 10
b = a (복사)
a = 20 (변경)
결과: a = 20, b = 10
-> 기본형은 값이 복사되어 독립적이다

=== 참조형 복사 실험 ===
dataA.value = 10
dataB = dataA (복사)
dataA.value = 20 (변경)
결과: dataA.value = 20, dataB.value = 20
-> 참조형은 같은 객체를 가리키므로 함께 변한다
```

---

## 힌트

- 기본형 변수: `int a = 10; int b = a;`
- 참조형 변수: `Data dataA = new Data(); Data dataB = dataA;`
- 기본형 복사: 값 자체가 복사됨
- 참조형 복사: 참조값(주소)이 복사됨 -> 같은 객체를 가리킴
