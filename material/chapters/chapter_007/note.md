## 노트 작성

> 수업 내용 정리. 자유롭게 작성.
- 자바의 클래스는 스택 자료구조처럼 실행
- 스택 영역은 참조값이 있고 실제 스택 구조로 메모리에 저장
- static 변수(클래스 변수)를 쓰면 static 영역에서 관리를 하기 때문에, 인스턴스 영역에서 관리가 아니기 때문에 해당 클래스의 인스턴스들 공통으로 사용 가능. 
    - 이런 클래스 변수와 인스턴스 변수를 합하여 멤버변수라고 통칭한다. 
- 지역변수는 스택 영역에 생성
- 인스턴스 변수는 힙 영역에 생성
- 클래스 변수는 메서드 영역의 static 영역에 보관, JVM이 로딩 되는 순간 생성
- static 변수는 인스턴스에서도 클래스에서도 접근이 됩니다. 
- static method는 인스턴스화를 거칠 필요 없이 바로 클래스에서 접근이 된다. 
- static method는 정적 변수만 접근할 수 있지만 그 외의 매서드는 모든 변수에 접근 가능하다. 
- 정적 매서드는 간단한 유틸함수 같은 게 많다. 
- static import을 해오면 클래스. 을 생략하고 static 매서드를 지정할 수 있다.
