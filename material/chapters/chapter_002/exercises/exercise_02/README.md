# 연습 문제 2: 참조형의 메서드 전달

## 난이도: 응용

## 학습 목표

- 메서드에 기본형을 전달할 때와 참조형을 전달할 때의 차이를 이해한다
- `public static void` 선언의 의미를 익힌다
- 참조형 매개변수를 통해 원본 객체를 수정할 수 있음을 이해한다

---

## 문제 설명

메서드에 기본형과 참조형을 각각 전달했을 때,
원본 값이 변경되는지 확인하는 프로그램을 작성하라.

**핵심 개념**:
- 기본형 전달: 값이 복사되어 전달됨 -> 메서드 내에서 변경해도 원본 불변
- 참조형 전달: 참조값이 복사되어 전달됨 -> 메서드 내에서 변경하면 원본도 변경

---

## public static void 설명

```java
public static void changePrimitive(int value) { ... }
```

| 키워드 | 의미 | Python 비교 |
|:-----|:-----|:-----|
| public | 어디서든 접근 가능 | 기본적으로 모든 함수가 public |
| static | 객체 생성 없이 호출 가능 | @staticmethod (self 없음) |
| void | 반환값 없음 | return이 없거나 None 반환 |

---

## 요구 사항

1. `Student.java`에 `Student` 클래스를 정의하라
   - name (String): 학생 이름
   - score (int): 점수

2. `Main.java`에서:
   - 기본형(int)을 메서드에 전달하고 변경 시도
   - 참조형(Student)을 메서드에 전달하고 변경 시도
   - 각각의 결과를 비교하여 출력

---

## 예상 출력

```
=== 기본형 전달 실험 ===
변경 전: number = 10
메서드 호출 (내부에서 20으로 변경 시도)
변경 후: number = 10
-> 기본형은 복사되어 전달되므로 원본 불변

=== 참조형 전달 실험 ===
변경 전: student.score = 60
메서드 호출 (내부에서 100으로 변경 시도)
변경 후: student.score = 100
-> 참조형은 같은 객체를 가리키므로 원본 변경됨
```

---

## 힌트

- 메서드 정의: `public static void 메서드명(타입 매개변수) { ... }`
- 기본형 매개변수: 값의 복사본을 받음
- 참조형 매개변수: 참조값의 복사본을 받음 (같은 객체를 가리킴)
