## 노트 작성

> 수업 내용 정리. 자유롭게 작성.
- private으로 하면 클래스 내에서만 접근 가능함.
- 적절한 제약이 있는 프로그램이 좋은 프로그램
- private : 모든 외부 호출을 막는다.
- default(package-private) : 같은 패키지 안에서의 호출만 허용한다.
- protected : 같은 패키지 안에서 호출 허용 + 상속 관계에서는 허용
- public : 모든 외부 호출을 허용한다. 

- 접근 제어자는 클래스, 필드, 생성자, 매서드에서만 사용할 수 있다. 
- 클래스 내의 매서드로 필드에 접근하는 것은 내부 호출로 들어간다.
