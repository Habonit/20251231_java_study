# 연습 문제 3: 참조값 이해

## 난이도: 심화

## 학습 목표

- 참조값(Reference)의 개념을 이해한다
- 변수에 객체가 아닌 **참조값**이 저장됨을 이해한다
- 같은 객체를 여러 변수가 참조할 수 있음을 이해한다

---

## 핵심 개념

> **"집은 하나인데 들어가는 문이 2개가 된 것이다."**
> - note.md에서 정리한 참조값 개념

```
┌─────────┐         ┌─────────────┐
│ 변수 a  │───────▶ │   객체      │
│  x001   │         │  name="홍"  │
└─────────┘    ┌──▶ └─────────────┘
               │
┌─────────┐    │
│ 변수 b  │────┘
│  x001   │
└─────────┘

a와 b는 같은 참조값(x001)을 가지므로 같은 객체를 가리킨다.
```

## 힌트

- `c2 = c1`은 **객체를 복사하는 것이 아니다**
- c1에 저장된 **참조값(주소)**을 c2에 복사하는 것이다
- 따라서 c1과 c2는 **같은 객체**를 가리키게 된다
