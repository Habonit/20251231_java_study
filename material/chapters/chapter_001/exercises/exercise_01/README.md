# 연습 문제 1: 클래스 정의와 객체 생성

## 난이도: 기초

## 학습 목표

- 클래스를 정의하는 방법을 익힌다
- 객체를 생성하고 값을 대입하는 방법을 익힌다
- `.` (dot) 연산자로 멤버 변수에 접근하는 방법을 익힌다

---

## 문제 설명

도서 관리 프로그램을 만들려고 한다.
`Book` 클래스를 정의하고, 책 정보를 저장한 후 출력하는 프로그램을 작성하라.

---

## 요구 사항

1. `Book.java`에 `Book` 클래스를 정의하라
   - 제목 (`title`): String
   - 저자 (`author`): String
   - 가격 (`price`): int

2. `Main.java`에서:
   - Book 객체 2개를 생성하라
   - 각 객체에 값을 대입하라
   - 정보를 출력하라

---

## 예상 출력

```
제목: 자바의 정석, 저자: 남궁성, 가격: 30000
제목: 이펙티브 자바, 저자: 조슈아 블로크, 가격: 36000
```

---

## 힌트

- 클래스 정의: `public class 클래스명 { ... }`
- 객체 생성: `new 클래스명()`
- 값 대입: `객체.멤버변수 = 값;`
