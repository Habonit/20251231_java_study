# 챕터 2: 기본형과 참조형

> 학습 날짜: 2026-01-02

## 핵심 개념

### 기본형 (Primitive Type)

변수에 **실제 값**이 직접 저장된다.

| 타입 | 크기 | 범위 |
|:----|:----|:----|
| `int` | 4byte | 약 -21억 ~ 21억 |
| `long` | 8byte | 매우 큰 정수 |
| `double` | 8byte | 실수 (소수점) |
| `boolean` | 1byte | true / false |

- 기본형은 자바가 기본으로 제공하는 데이터 타입이다.
- 개발자가 새로 정의할 수 없다.

### 참조형 (Reference Type)

변수에 **객체의 참조값(주소)**이 저장된다.

```java
Student student1 = new Student();  // student1에는 참조값이 저장됨
```

- 개발자가 클래스로 직접 정의할 수 있다.
- `String`은 클래스(참조형)이지만, 기본형처럼 사용할 수 있게 편의 기능을 제공한다.

### 변수의 종류

| 종류 | 선언 위치 | 설명 |
|:----|:--------|:----|
| 멤버 변수 | 클래스 내부 | 클래스에서 선언한 변수 |
| 지역 변수 | 메서드 내부 | 메서드나 블록 안에서 선언한 변수 |

---

## 가비지 컬렉션 (GC)

아무도 참조하지 않는 객체를 메모리에서 자동으로 제거하는 기능.

```java
Student s = new Student();
s = null;  // 기존 객체는 이제 참조되지 않음 → GC 대상
```

---

## 객체 접근

객체의 멤버에 접근할 때는 **dot(.)** 을 사용한다.

```java
student1.name = "홍길동";
student1.age = 20;
```
