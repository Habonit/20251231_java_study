# 연습 문제 2: 증감 연산자 이해

## 난이도: 기초

## 학습 목표

- 증감 연산자 (++, --)의 사용법을 익힌다
- 복합 대입 연산자 (+=, -=)와의 차이를 이해한다
- 전위/후위 증감 연산자의 차이를 이해한다

---

## 핵심 개념

### 증감 연산자

| 연산자 | 예시 | 동일한 표현 |
|:-----|:-----|:-----|
| `++` | `volume++` | `volume = volume + 1` |
| `--` | `volume--` | `volume = volume - 1` |
| `+=` | `volume += 5` | `volume = volume + 5` |
| `-=` | `volume -= 3` | `volume = volume - 3` |

### 전위(prefix) vs 후위(postfix)

```java
int a = 5;
int b = ++a;  // 전위: 먼저 증가 후 대입 → a=6, b=6

int x = 5;
int y = x++;  // 후위: 먼저 대입 후 증가 → x=6, y=5
```

| 구분 | 문법 | 동작 순서 |
|:-----|:-----|:-----|
| 전위 (prefix) | `++a` | 증가 → 사용 |
| 후위 (postfix) | `a++` | 사용 → 증가 |

## 예상 출력

```
=== 기본 증감 연산자 ===
초기값: volume = 0
volume++ 후: 1
volume++ 후: 2
volume-- 후: 1

=== 복합 대입 연산자 ===
초기값: count = 10
count += 5 후: 15
count -= 3 후: 12

=== 전위 vs 후위 ===
a = 5
b = ++a 후: a=6, b=6 (먼저 증가 후 대입)

x = 5
y = x++ 후: x=6, y=5 (먼저 대입 후 증가)
```

---

## 힌트

- `volume++`와 `volume += 1`은 같은 결과
- 전위(++a): 증가 먼저, 사용 나중
- 후위(a++): 사용 먼저, 증가 나중
